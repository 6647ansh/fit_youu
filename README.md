# Java Diet Project

-FitTrack
Overview

FitTrack is a console-based Java application designed to help users manage their diet and fitness routines in a simple and interactive way. Users can track calorie intake, get personalized meal plans, log exercises, and monitor progress over time. All data is stored in a database, making it easy to keep track of fitness goals.

Features

User Accounts – Create an account, log in, and manage your profile.

Personalized Diet Plans – Get meal suggestions based on your weight, height, BMI, and fitness goals.

Calorie Tracking – Monitor your daily calorie intake and stay on track.

Fitness Logging – Record exercises, steps, and calories burned.

Database Storage – All user info, diet plans, and fitness logs are saved in a SQL database.

Weekly Progress Reports – See your weight changes and achievements over time.

Reminders – Get alerts for meals, water intake, and workouts.

Interactive Console – Easy-to-use command-line interface for smooth navigation.

Getting Started
Prerequisites

Java Development Kit (JDK) installed

IDE like Eclipse or IntelliJ IDEA

PostgreSQL database (set up and configure connection in the project)

Installation

Clone the repository:

git clone https://github.com/brahmibora/FitTrack.git


Open the project in your IDE and configure the database connection.

Build and run the project to start using FitTrack.

Usage

Run the application and follow the prompts to create an account or log in.

Enter your personal information and fitness goals to generate diet and exercise plans.

Navigate through menus to track calories, exercises, and weekly progress.

Use reminders to stay on top of meals, water intake, and workouts.
